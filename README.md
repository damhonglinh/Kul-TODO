TODO-List-Android
=================

An TODO List application on Android that can sync with Google Task.<br/><br/>
Features:
<ul>
  <li>Create, edit, delete a task.</li>
  <li>Mark as read.</li>
  <li>Group tasks.</li>
  <li>Sort tasks.</li>
  <li>Set 3 priorities.</li>
  <li>Add phone contacts to a task.</li>
</ul>
